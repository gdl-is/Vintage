Installare Windows 3.1 su DOSBox

Guida passo-passo per configurare S3Trio, Sound Blaster, Mouse e Tastiera

1. Preparazione dei File

Prima di iniziare, crea una cartella sul tuo PC (es. C:\dosbox_c) che fungerà da Hard Disk C: temporaneo per l'emulatore.

  • Crea una sottocartella C:\dosbox_c\win31_install e copia qui i file di installazione estratti dai dischetti di Windows 3.1.
  • Crea una sottocartella C:\dosbox_c\s3driver dove andrai a inserire i driver per la scheda video.

5. Installazione Driver Video S3 Trio

Al termine dell'installazione, chiudi Windows 3.1 per tornare al prompt di DOS (C:\>) e segui questi passi:

  1. Digita cd windows e poi setup.exe (avvierai il setup di configurazione in modalità testo).
  2. Sposta il cursore su Display e premi Invio.
  3. Scorri fino in fondo alla lista, seleziona Other (Requires disk...) e premi Invio.
  4. Inserisci il percorso dei driver S3 estratti in precedenza: C:\s3driver e premi Invio.
  5. Seleziona la risoluzione desiderata (scelta consigliata per DOSBox: 800x600 256 Colors (Small Fonts)).
  6. Conferma premendo Invio per copiare i file, salvare e uscire.

6. Configurazione Audio (Sound Blaster 16)

Windows 3.1 include già i driver nativi per la Sound Blaster. All'interno dell'interfaccia di Windows:

  1. Apri il gruppo Main (Principale) e fai doppio clic su Control Panel (Pannello di Controllo).
  2. Apri la voce Drivers e clicca sul pulsante Add (Aggiungi).
  3. Seleziona Creative Labs Sound Blaster 16 Or AWE32 dalla lista proposta.
  4. Configura i parametri hardware inseriti nel file .conf: Port: 220, IRQ: 7, DMA: 1.
  5. Riavvia Windows 3.1 per applicare i driver audio.

    10. Risoluzione Problemi e Suggerimenti

    Cattura e Sblocco del Mouse: Se il cursore del mouse scappa dalla finestra di DOSBox o non è sincronizzato, premi la combinazione di tasti CTRL + F10 per bloccarlo permanentemente dentro l'emulatore. Premi la stessa combinazione per liberarlo.
    Audio Gracchiante o Rallentato: Se la Sound Blaster gracchia, aumenta i cicli della CPU di DOSBox premendo CTRL + F12 durante l'emulazione, oppure imposta cycles=max nella sezione [cpu] del tuo file di configurazione.
    Layout Tastiera Italiana: Se i caratteri speciali non corrispondono alla tua tastiera, assicurati di inserire il comando keyb it subito prima di avviare Windows (nella sezione autoexec).
